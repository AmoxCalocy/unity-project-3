# Captain Blaster

Captain Blaster 是一款基于 Unity 引擎开发的 2D 垂直滚动射击游戏。本项目为虚拟现实开发引擎课程的实验项目。

## 游戏简介

玩家将驾驶一艘宇宙飞船在太空中飞行，通过发射子弹摧毁迎面而来的流星以获取分数。流星会随着时间的推移不断且越来越快地出现，直到玩家的飞船被流星击中，游戏结束。

## 核心玩法
*   **操作方式**：玩家通过键盘左右方向键控制飞船的水平移动，按下空格键发射子弹。
*   **得分机制**：每摧毁一颗流星，玩家将获得 1 点分数。
*   **无限滚动**：利用两张背景图像交替循环移动，实现无限深空飞行的视觉效果。
*   **难度递增**：流星的生成频率会随着游戏时间的推移而逐渐增加。

## 主要组件与功能

本项目的主要系统划分为以下几个核心部分：

*   **游戏控制 (GameManager)**：负责整体游戏状态的管理，包括分数的统计与 UI 显示、玩家生命状态的监控以及游戏结束画面的呈现。
*   **玩家控制 (ShipController)**：处理飞船的移动输入、射击冷却时间的控制以及与流星碰撞检测的逻辑。
*   **敌人与生成 (MeteorMoverScript, MeteorSpawnScript)**：控制流星的向下飞行、随机自转效果，并在屏幕顶部随机位置动态生成流星预制体。
*   **武器系统 (Bullet)**：控制子弹的向上飞行轨迹及与流星的碰撞销毁逻辑。
*   **环境系统 (BackgroundScript, DestoryOnTrigger)**：管理星空背景的无缝滚动，并清理飞出屏幕外的子弹和流星实体以优化性能。

## 开发环境
*   Unity 引擎 (2D 模式)

双击re3.exe以运行游戏。
